INSTALL NODEJS AND NPM ON YOUR SYSTEM

after you have that setup:
double click the run.sh file in the folder

If you want to change the amount generated do to:
The bottom line of the code in index.js and change the number

Files will output to combined/Images


To Adjust Rarity:
End filename with A / B / C / D / S
S - 3%
A - 7%
B - 20%
C - 30%
D - 40%

Files not ending with ranking will just default to D
